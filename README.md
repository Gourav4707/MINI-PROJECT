# 🚀 Mini Projects Collection (Python)

A collection of beginner-to-intermediate level Python projects built to strengthen core programming concepts, problem-solving, and real-world application design.

---

## 📌 Projects Included

### 🏧 ATM System

A command-line based ATM simulation that allows users to:

* Check account balance
* Deposit money
* Withdraw money
* Perform basic banking operations securely

---

### 🎮 Game Module

A simple interactive game system demonstrating:

* Game logic implementation
* User interaction via CLI
* Modular code structure

---

### 📚 Library Management System

A basic system to manage library operations:

* Add and remove books
* Issue and return books
* Track available inventory

---

## 🛠️ Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Functions & Modular Programming
  * File/Folder Structure (Clean Architecture)
  * Basic OOP Principles
  * CLI-based Interaction

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Gourav4707/MINI-PROJECT.git
```

2. Navigate into the project folder:

```bash
cd MINI-PROJECT
```

3. Run any module:

```bash
python ATM/main.py
python GAME/main.py
```
